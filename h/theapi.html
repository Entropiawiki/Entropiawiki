EntropiaWiki API Usage Guide

The EntropiaWiki API provides endpoints to retrieve data related to planets, mobs, societies, loot, vehicles, armor, continents, locations, and weapons from the Entropia Universe. This guide outlines the available endpoints and provides instructions on how to use them.

Base URL: https://entropiawiki.co.uk/api/

  1. Fetch Planet Data

    Endpoint: https://entropiawiki.co.uk/api/planets

    Method: GET

    Description: Fetches information about planets in the Entropia Universe.

  2. Fetch Weapon Data

    Endpoint: https://entropiawiki.co.uk/api/weapon

    Method: GET

    Description: Retrieves information about weapons and their attributes.

Example Usage:

Using Fetch in JavaScript:

fetch('https://entropiawiki.co.uk/api/planets')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Using Axios in JavaScript:

axios.get('https://entropiawiki.co.uk/api/mobs')
    .then(response => console.log(response.data))
    .catch(error => console.error('Error:', error));

Please ensure you have proper permissions and adhere to the terms of use outlined by EntropiaWiki.co.uk for accessing and using their API.
